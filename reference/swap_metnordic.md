# METNordic input for SWAP

This function converts Meteo data from MET Nordic into a SWAP-compatible
format. Note: need to use `metnordic_csv()` to create proper input for
this function.

## Usage

``` r
swap_metnordic(dldir, name, outpath, timescale = "daily", verbose = FALSE)
```

## Arguments

- dldir:

  download directory containing data and metadata, as generated by
  [`metnordic_extract()`](https://moritzshore.github.io/miljotools/reference/metnordic_extract.md)

- name:

  name of the extracted point, as defined in
  [`metnordic_extract()`](https://moritzshore.github.io/miljotools/reference/metnordic_extract.md)

- outpath:

  desired location for the swap .met file to be written

- timescale:

  "daily" or "hourly". (hourly not supported yet)

- verbose:

  logical, print some stuff to console

## Value

path to written .met file
