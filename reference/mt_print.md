# custom print function for miljotools

custom print function for miljotools

## Usage

``` r
mt_print(verbose, function_name, text, text2 = NULL, rflag = FALSE)
```

## Arguments

- verbose:

  print or not

- function_name:

  function name string

- text:

  string 1

- text2:

  string 2

- rflag:

  in place replacement flag

## Value

nothing
